# 개요
컴프입 팀프 프로그램 제작 및 기여 공유 레포지토리

### 1. 프로젝트 개요

- **프로젝트명:** 키스트로크 다이내믹스
- **팀원 구성:[팀원 및 역할](https://www.notion.so/369d0329137280abb187e8c1e01bca86?pvs=21)**
- **한 줄 요약:** 비밀번호 키 뿐만 아니라 타이핑 패턴까지 대조하는 강화된 보안 시스템

### 2. 개발 배경 및 필요성

- **해결하려는 실생활 문제**
    - 비밀번호 입력 과정이 노출됨
        
        외부에서 로그인 시, 주변 사람의 시선이나 CCTV 같은 감시 장치로 인해 자신의 비밀번호가 쉽게 노출되는 단점이 있다. 
        
    - 비밀번호와 아이디를 알면 로그인할 수 있음
        
        보통 입력창에 아이디는 공개되어 있고, 비밀번호는 가려져 있지만 누군가 비밀번호를 아는 경우 쉽게 로그인할 수 있다.
        
- **기존 서비스와의 차별성**
    - 키(비밀번호) 뿐만 아니라 타이핑 패턴 또한 검사함
        
        타이핑 패턴을 알아도 키를 모르면 로그인할 수 없고, 키를 알아도 타이핑 패턴을 모르면 로그인할 수 없음. 즉, 로그인에 필요한 정보가 하나 더 추가됨.
        
    - 더미 패턴 입력 지원
        
        비밀번호를 입력하기 전후로 아무 글자나 마구 타이핑해도, 시스템이 진짜 타이핑 패턴만 식별할 수 있으므로 보안성이 강화됨.
        
    - 로그인 실패시 징벌 부과
        
        일정 시간 내에 올바른 패턴이 감지되지 않거나, 오입력 누적이 기준치를 초과할 경우 보안 위협으로 간주하여 패턴 입력 시간의 상한이 더 짧아짐.
        

### 3. 주요 기능 및 시스템 구조

- **핵심 기능**
    - 비밀번호 입력과 동시에 타이핑 패턴 입력
    - 아이디 & 비밀번호 & 타이핑 패턴 검증 후 실패시 징벌 부과
    - 더미 패턴 속 진짜 패턴 식별 지원
    - 비밀번호 변경 및 이전 비밀번호 내역 확인
    - 이전 비밀번호의 패턴 재사용 배제
